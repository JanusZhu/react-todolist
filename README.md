# Todolist

A todolist React app.

[Live Demo](https://januszhu.github.io/react-todolist/) :point_left:

## Getting started

```
git clone https://github.com/JanusZhu/react-todolist.git
cd react-todolist
npm install
npm start
```

## Built with

- [React](https://reactjs.org/)
- [Tailwind](https://tailwindcss.com/)
